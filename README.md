# Sistema de Restaurante en Python

Este programa fue desarrollado en Python y permite gestionar un menú de restaurante utilizando matrices, funciones, ciclos y condicionales.

Funciones principales:
- Ver menú de productos
- Mostrar promociones por día
- Aplicar descuentos automáticos
- Agregar productos a factura
- Eliminar productos
- Mostrar factura final

Tecnologías utilizadas:
- Python
- GitHub
